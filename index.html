nText

a local text editor

nText is a simple app written in JavaScript for the NodeJS platform.

It has been packaged for the Mac desktop using Electron.

The nText text creation app is powered by web technologies including the ExpressJS server, the AngularJS framework and the NeDB database.

Looking for an exit strategy for your content? You can highlight and drag content to and from Sublime Text!
